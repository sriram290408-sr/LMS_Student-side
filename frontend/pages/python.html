Installation & Setup

Let's get Python installed on your computer and set up your development environment.

Step 1: Download Python

Visit the official Python website and download the latest version: 

1. Go to https://www.python.org/downloads/
2. Click "Download Python 3.x.x" (latest version)
3. Run the installer
4. IMPORTANT: Check "Add Python to PATH"
5. Click "Install Now"

Step 2: Verify Installation

Open your terminal or command prompt and verify Python is installed: 

python --version
# Should display: Python 3.x.x

python
# Opens Python interactive shell
>>> print("Hello, Python!")
Hello, Python!
>>> exit()
